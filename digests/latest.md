# X AI 博主精选简报

**生成时间**: 2026-01-30 | **状态**: 等待首次运行

## 📖 说明

这是一个自动化的 AI 博主内容精选系统，每天抓取 X(Twitter) 上 20 位优质 AI 博主的高质量推文，使用 Claude AI 生成结构化简报。

### ✨ 功能特性

- ⏰ **自动运行** - 每天 07:00 自动生成
- 🐦 **精选博主** - 20 位 AI 领域优质博主
- 🤖 **AI 分析** - Claude AI 智能筛选和分类
- 💬 **微信推送** - PushPlus 自动推送到微信

### 📊 内容板块

- 🛠️ AI工具精选
- ⚙️ AI工作流
- ✍️ 提示词工程
- 💻 AI编程实践
- 📝 内容创作
- 🧠 AI思考与判断力

### 🚀 使用方法

首次运行需要配置 GitHub Secrets：

1. **BIRD_AUTH_TOKEN** - Bird CLI Auth Token
2. **BIRD_CT0** - Bird CLI CT0 Token
3. **ANTHROPIC_API_KEY** - Claude API Key
4. **PUSHPLUS_TOKEN** - 微信推送 Token (可选)

配置完成后，系统会在每天 07:00 自动运行。
